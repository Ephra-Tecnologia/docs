| Evento | Disparado quando |
| --- | --- |
| `transfer_created` | Uma transferência é iniciada. |
| `transfer_completed` | A transferência é concluída com sucesso. |
| `transfer_canceled` | A transferência é cancelada. |
| `transfer_updated` | A transferência muda de status. |

Todos os eventos `transfer_*` compartilham o mesmo formato de payload.

## Payload

<Tabs>

    {
      "id": "9f1c8e2a-2b7d-4c1a-9f3e-uuid",

      "event": "transfer_completed",
      "scope": "user",
      "transfer": {
        "id": "clxtransfer123",
        "amount": 50000,
